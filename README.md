# To-Do-List

A simple and elegant to-do list application built with vanilla JavaScript, CSS, and HTML. Organize your tasks efficiently with an intuitive user interface.

## 🎯 Features

- ✅ **Add Tasks** - Easily create new to-do items
- ✏️ **Edit Tasks** - Modify existing tasks
- 🗑️ **Delete Tasks** - Remove completed or unwanted tasks
- ✔️ **Mark Complete** - Check off tasks as you complete them
- 💾 **Local Storage** - Your tasks persist even after closing the browser
- 🎨 **Clean UI** - Minimalist and user-friendly interface
- 📱 **Responsive Design** - Works seamlessly on desktop and mobile devices

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No dependencies or installation required

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/manivarma701-jpg/To-Do-List.git
   cd To-Do-List
   ```

2. Open `index.html` in your web browser:
   ```bash
   open index.html
   ```
   Or simply double-click the `index.html` file.

## 🎮 How to Use

1. **Add a Task**: Type your task in the input field and press Enter or click the "Add" button
2. **Complete a Task**: Click the checkbox next to the task to mark it as complete
3. **Edit a Task**: Click the edit button to modify the task text
4. **Delete a Task**: Click the delete button to remove the task from your list

## 📝 Features Breakdown
